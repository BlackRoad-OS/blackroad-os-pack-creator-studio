## Usage

1. Copy the template to your project
2. Fill in the bracketed placeholders `[...]`
3. Remove or modify sections as needed
4. Submit for review when ready

## Template Design Principles

- All templates follow the schemas in `/schemas/`
- Placeholders are clearly marked with `[brackets]`
- Sections can be removed if not applicable
- Risk and approval checkboxes are pre-included
- Brand reference fields link to `blackroad-os-brand`
